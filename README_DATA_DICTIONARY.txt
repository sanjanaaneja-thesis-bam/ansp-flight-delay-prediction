WEATHER DATA DICTIONARY
======================================================================

Source: Iowa Environmental Mesonet (IEM)
URL: https://mesonet.agron.iastate.edu/
Download Date: 2026-03-07 13:14:10
Study Period: 2023-06-01 to 2023-07-31

VARIABLES:
----------------------------------------------------------------------
day                  - Date (YYYY-MM-DD)
station              - IATA station code (3-letter)
max_temp_f           - Maximum Air Temperature [F]
min_temp_f           - Minimum Air Temperature [F]
max_dewpoint_f       - Maximum Dew Point [F]
min_dewpoint_f       - Minimum Dew Point [F]
precip_in            - Daily Precipitation [inch]
avg_wind_speed_kts   - Average Wind Speed [knots]
snow_in              - Reported Snowfall [inch]
avg_feel             - Average 'Feels Like' Temperature [F]
IATA                 - 3-letter IATA airport code

NOTES:
----------------------------------------------------------------------
- Missing values appear as blank
- Precipitation values of 0.0001 represent trace amounts
- Data is based on local calendar day for each station
